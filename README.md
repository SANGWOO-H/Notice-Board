# Notice-Board
PHP 언어를 이용하여 게시판을 만들었습니다.

## 기능
1. 로그인, 회원가입, 로그아웃 기능
2. 자유게시판에서 자유롭게 글을 쓸 수 있다.
3. 조회수 확인
